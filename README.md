## 网页本地部署：
#### 1.一键安装库
进入requirements.txt所在的目录，执行命令：

```pip install -r requirements.txt```
#### 2.启动本地服务器
进入项目根目录，执行命令：

```python manage.py runserver```
#### 3.在浏览器输入：localhost:8000/即可进入网页
